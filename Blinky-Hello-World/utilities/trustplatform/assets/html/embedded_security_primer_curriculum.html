TRAINING VIDEOS

To help your design journey, Microchip is offering this library of security videos. These videos are presented in the order of natural progression. We invite you to watch them in the proposed sequence. They provide an introduction of numerous concepts that needs to be well understood to adequately secure a design.

Cryptography Primer (part 1)

Cryptography Primer (part 2) : Confidentiality, Integrity, Authentication

During this tutorial about embedded security, Microchip discusses the concepts of 3 key pillars of security which are Authentication, Integrity and Confidentiality.

Cryptography Primer (part 3) : Hashing

Learn about the basics of embedded security in this Youtube video where Microchip discusses the concepts of hashing, how and when to use hashing.

Cryptography Primer (part 4) : Asymmetric Authentication

During this tutorial about embedded security, Microchip discusses the concepts of Asymmetric Cryptography and illustrate how authentication can be implemented highlighting the importance of protecting private keys in hardware secure key storage.

Cryptography Primer (part 5) : Chain of Trust

In the chain of trust session of the Microchip cryptography primer tutorial, Microchip discusses concepts especially relevant to understand how to implement a robust authentication between a host and a client using public key infrastructure (PKI).
